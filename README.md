# Zoyla Landing Page

Landing page for [Zoyla](https://github.com/behnamazimi/zoyla) — a fast, lightweight HTTP load testing desktop app built with Rust.

**Live:** [zoyla.app](https://zoyla.app)

## Development

```bash
npm install
npm run dev
```

## Scripts

- `npm run dev` — Start development server
- `npm run build` — Build for production
- `npm run lint` — Lint with oxlint
- `npm run format` — Format with Prettier
